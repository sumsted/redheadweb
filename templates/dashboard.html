Help

Create a set of lines that identify paths that a robot may take. To create a line, you will identify start and end points then create a line. You may select existing points and lines. You may delete lines and points, but only with the hotkeys.

Steps

  1. Position and zoom map on area to be worked or load existing map.
  2. Click on map to create a start point.
  3. Click on map to create an end point.
  4. Click on Add Line button to link points.
  5. Repeat steps 3 and 4 to create a continuing path. Select existing points as start points to create branches.
  6. Name your map with a map key and SAVE it. Very important SAVE it.

Hotkeys

  • Help (Control - h)
  • Create Line (Control - d)
  • Delete Point (Control - s)
  • Delete Line (Control - x)
